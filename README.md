# Solana Exit Dashboard – Final with RSI Divergence

Includes:
✅ Weekly RSI divergence logic  
📱 Mobile UI  
🔐 Auth  
🚨 Scoring + confidence model

## Setup

1. Install dependencies: `npm install`
2. Add `.env.local`:
```
GITHUB_ID=your_id
GITHUB_SECRET=your_secret
NEXTAUTH_SECRET=random_string
```
3. Start: `npm run dev`
4. Deploy to Vercel
